Description

Mage is a console-based Java application that simulates a potion crafting system. Users manage an inventory of magical ingredients and combine them to create various potions. The program applies object-oriented programming concepts to model ingredients, potions, and crafting logic while providing interactive user input and feedback.

Skills Learned

  • Java programming fundamentals
  • Object-oriented design (classes, inheritance, encapsulation)
  • Data management using collections
  • User input handling
  • Program logic and flow control
  • Debugging and testing
